# Quick Reference Guide - Kuiper Partner Onboarding Hub

## Common Tasks

### For PAMs (Partner Account Managers)

#### Create a New Partner
1. Navigate to Dashboard
2. Click "Add New Partner"
3. Fill in partner details:
   - Partner Name
   - Contract Type (PPA, Distribution, Sales-Agent, Other)
   - Tier (0, 1, or 2)
   - CCV (Contractually Committed Value)
   - Target Launch Date
4. Assign team members (PDM, TPM, PSM, TAM)
5. Click "Create Partner"

#### Submit Pre-Contract Questionnaire
1. Navigate to partner detail page
2. Click "Pre-Contract PDM Engagement"
3. Complete all 5 sections:
   - Executive Sponsorship
   - Commercial Framework
   - Technical Feasibility
   - Timeline (must be within 60 days)
   - Strategic Classification (Tier 0/1)
4. Review section statuses (pass/fail)
5. Sign the questionnaire
6. Click "Submit"

#### Track Partner Progress
1. View Dashboard for overview of all partners
2. Partners are grouped by current gate
3. Click partner name to see detailed progress
4. Check gate timeline and completion status
5. View upcoming milestones

### For PDMs (Partner Development Managers)

#### Complete Gate 0 Questionnaire
1. Navigate to partner detail page
2. Click "Gate 0: Onboarding Kickoff"
3. Complete all 6 sections:
   - Contract Execution
   - Partner Team
   - Launch Timing
   - Financial Bar
   - Strategic Value
   - Operational Readiness
4. Note: Tier 0 partners (CCV â‰¥ $50M) automatically qualify
5. Sign and submit

#### Complete Gate 1 Questionnaire
1. Navigate to partner detail page
2. Click "Gate 1: Ready to Sell"
3. Complete all 3 phases:
   - Phase 1A: Kickoff & Planning (Weeks 1-3)
   - Phase 1B: GTM & Discovery (Weeks 3-6)
   - Phase 1C: Training (Weeks 7-12)
4. Ensure all criteria are met:
   - Project plan approved
   - GTM strategy approved
   - Technical architecture defined
   - Sales team certified
5. Sign and submit

#### Access Documentation
1. Click "Documentation" in navigation
2. Filter by gate (Pre-Contract, Gate 0, Gate 1)
3. Filter by phase (1A, 1B, 1C)
4. Click links to access resources
5. Use search to find specific topics

### For TPMs (Technical Partner Managers)

#### Complete Gate 2 Questionnaire
1. Navigate to partner detail page
2. Click "Gate 2: Ready to Order"
3. Complete both phases:
   - Phase 2A: Systems Integration (Weeks 13-17)
   - Phase 2B: Operational Process Setup (Weeks 13-17)
4. Verify all criteria:
   - API integration complete
   - Monitoring active
   - Test transactions successful
5. Sign and submit

#### Review Technical Requirements
1. Navigate to Documentation
2. Filter by "Gate 2"
3. Access API integration guides
4. Review monitoring setup documentation
5. Check operational process templates

### For PSMs/TAMs (Partner Success/Technical Account Managers)

#### Complete Gate 3 Questionnaire
1. Navigate to partner detail page
2. Click "Gate 3: Ready to Deliver"
3. Complete both phases:
   - Phase 3A: Operational Readiness (Weeks 18-19)
   - Phase 3B: Launch Validation (Week 20)
4. Verify all criteria:
   - Beta testing successful
   - Support transition complete
   - Operational metrics validated
5. Sign and submit

#### Monitor Post-Launch Partners
1. View Dashboard
2. Filter by "Post-Launch" gate
3. Review operational metrics
4. Check for any issues or blockers
5. Access post-launch documentation

### For Admins

#### View All Partners
1. Navigate to Dashboard
2. Admin view shows all partners regardless of ownership
3. Use filters to narrow down:
   - By gate
   - By tier
   - By team member
   - By date range

#### Generate Reports
1. Navigate to Reports page
2. View summary statistics:
   - Total partners
   - Partners by gate
   - Average time per gate
   - Gate pass/fail rates
3. Filter by date range, gate, tier, or team member
4. Click "Export to CSV" to download data

#### Manage User Roles
1. Access Netlify Identity dashboard
2. View all users
3. Assign roles (PAM, PDM, TPM, PSM, TAM, Admin)
4. Update user permissions
5. Deactivate users if needed

## Keyboard Shortcuts

- `Ctrl/Cmd + K` - Search documentation
- `Ctrl/Cmd + S` - Save draft (auto-save also active)
- `Esc` - Close modal/dialog
- `Tab` - Navigate between form fields
- `Enter` - Submit form (when on submit button)

## Best Practices

### Questionnaire Completion
1. **Save frequently** - Auto-save is active, but manual saves are good practice
2. **Review before submitting** - Check all sections for accuracy
3. **Document decisions** - Use notes fields to explain choices
4. **Collaborate** - Discuss with team members before submission
5. **Keep documentation handy** - Reference guides while completing

### Gate Progression
1. **Don't rush** - Ensure all criteria are genuinely met
2. **Communicate blockers** - Report issues early
3. **Update regularly** - Keep partner status current
4. **Document learnings** - Note what worked and what didn't
5. **Celebrate milestones** - Acknowledge gate completions

### Data Quality
1. **Be accurate** - Enter correct information
2. **Be complete** - Fill all required fields
3. **Be consistent** - Use standard formats
4. **Be timely** - Update information promptly
5. **Be clear** - Write clear notes and comments
